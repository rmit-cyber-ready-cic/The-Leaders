You have been a victim of email phishing attack


Fake e-mail or phishing e-mails are the mail, making people believe that it comes from a legitimate source to steal personal information.


Countermeasures for Phishing attacks:

  • Network-level protection: This level of protection help us to restrict the IP addresses to enter the network, the most common name is the blacklist filter.
    • Anti-spam filters: Bulk e-mails are a potential threat. This technique helps us to identify the origin of the e-mail. These e-mails contain the false address of the user and information, which looks like it comes from a trusted source.
    • DNS- based blacklist: This blacklist uses DNS protocols. The DNS- blacklist requires attention from the internet service provider to update the list because it is reactive.
  • Authentication: The authentication approach for both user and the server checks if the attacker pretends to be a legitimate sender; it is best user level and domain level.
  • Classification of feature-based e-mail: Attackers embed the link in the e-mail, which ask the user to disclose the private data. It is the most common way which the attackers use to steal the data.
    • Link Feature: The link guard algorithm is used to identify the fake links in the e-mails, the actual link mentioned in the text used by the attackers to lure the users.
    • Structural Features: To classify the e-mails before it reaches the user support vector machine is used to classify phishing e-mails.

    Prevention of Phishing attacks

  • Checking the grammar mistakes is very helpful to find an unnatural link, which helps us keep our data safe (Mario Ciprian Birlea 2020).
  • Containing the e-mail addresses could tell you about the source of the e-mail. Sometimes attacker uses the numeric number in the e-mail addresses, which shows that it is not from the trusted site (Mario Ciprian Birlea 2020).
  • The big organisations are rarely calling them with a generic greeting (Mario Ciprian Birlea 2020).
  • Some attackers send the mail, asking for urgent actions, such as updating the details by clicking the link; otherwise, their account will be disabled (Mario Ciprian Birlea 2020).
